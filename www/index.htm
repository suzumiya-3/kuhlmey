概要

 このサービスはDiscordのOauth2機能を使って、 サーバーのメンバーを記録し、もし荒らされた等でサーバーを作り直すときに、
メンバーを復元をすることができます。 更にはreCAPTCHA機能を持っているので、bot対策としても運用が可能です。

技術情報

 開発言語: Python
概要: Oauth2を使用しつつ、Flaskでの認証機能の公開。
バージョン: 1.0.0
ライセンス: MIT

起動方法

 このサービスはソースコードを公開しているため、個人での運用が可能です。
1. サーバーにPython3環境を用意する
2. ソースコードをダウンロードする (右側参照)
3. コンフィグを設定する(下の段の“コンフィグ”参照)
4. python main.py等で起動させる。
